# Điều khiển

## Remote Xiaomi / Android TV

| Nút | Menu | Trong ván |
|-----|------|-----------|
| D-pad | Chọn mục | Di chuyển con trỏ ô |
| OK | Vào mục | Chọn quân → chọn ô đích |
| Back | Về trước | Hủy chọn / về menu |
| H (bàn phím) | — | Gợi ý Dorami |
| U (bàn phím) | — | Hoàn tác |

## Luồng đi quân

1. Đưa khung vàng tới quân phe đang đánh.  
2. OK → ô xanh = đi, ô cam = ăn.  
3. Đưa khung tới ô đích → OK.  
4. Phong cấp: ← → chọn Xuka/Chaien/Xeko/Nobita → OK.

## Học chơi (tutorial)

Menu → **Học chơi (5 bước)** → ← / → hoặc OK để chuyển bước · Back về menu.

## Âm thanh

Cài đặt: bật/tắt **Âm thanh hiệu ứng** và **Nhạc nền**.
